# Commande /test-lolve

Teste lolve-cartography sur le projet LOLVE.

## Instructions

### Index complet
```bash
node bin/cli.cjs --project C:/lolve annotate index
```

### Scan rapide
```bash
node bin/cli.cjs --project C:/lolve annotate scan
```

### Statistiques
```bash
node bin/cli.cjs --project C:/lolve annotate stats
```

## Arguments

- `/test-lolve` ou `/test-lolve index` : Génère l'index
- `/test-lolve scan` : Scanne la codebase
- `/test-lolve stats` : Affiche les statistiques
- `/test-lolve enrich` : Enrichit avec métadonnées
